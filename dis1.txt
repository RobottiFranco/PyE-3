Para usar es necesario instalar las siguientes librerías:

pip install seaborn
pip install matplotlib
pip install scipy
pip install numpy


Explicación del código:

1. Generación de muestras aleatorias: Utilizamos `binom.rvs` de `scipy.stats` para generar muestras aleatorias de tamaños \( 10^2, 10^3, 10^4 \) y \( 10^5 \).
2. Diagrama de cajas: Usamos `seaborn.boxplot` para crear diagramas de cajas de las muestras generadas.
3. Histogramas: Utilizamos `matplotlib.pyplot` para crear histogramas de las muestras.
4. Cálculo de la mediana y la moda: Utilizamos `numpy.median` para calcular la mediana y `scipy.stats.mode` para calcular la moda de cada muestra.
5. Cálculo de la media empírica y comparación con la esperanza teórica: Calculamos la media empírica de cada muestra y la comparamos con la esperanza teórica de la distribución binomial.
6. Cálculo de la varianza empírica y comparación con la varianza teórica: Calculamos la varianza empírica de cada muestra y la comparamos con la varianza teórica de la distribución binomial.
